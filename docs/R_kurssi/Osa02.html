2.1 Apply, quantile & aggregate tehtävä

Tehtävänanto:

Tee skripti, joka laskee alla olevasta aineistosta aggregate-funktiolla ryhmien keskiarvoiät, quantile-funktiolla WM-suorituksen 90 persentiilin, ja apply-funktiolla Age ja WM muuttujien keskiarvon


Ratkaisu:

data <- data.frame(
  Group = as.factor(rep(c("GroupA","GroupB","GroupC"),10, each=10)), 
  Age = as.numeric(sample(18:65, 300, replace=TRUE)),
  WM =  as.numeric(sample(1:100, 300, replace=TRUE))
)

aggregate(Age ~ Group, data = data, FUN = mean)
   Group   Age
1 GroupA 40.95
2 GroupB 42.53
3 GroupC 40.10
quantile(data$WM, probs = seq(0, 1, 0.9))
 0% 90% 
  1  88 
apply(data[,2:3], 2, mean) 
     Age       WM 
41.19333 48.62000

Ensiksi luotiin datafreimi data.frame() funktiolla

Aggregate funktiossa ensiksi valitaan numeerinen freimi datasetistä, josta keskiarvo (tai jokin muu) lasketaan, sitten ~ merkin jälkeen # ryhmittelevä tekijä, eli se minkä mukaan data jaotellaan. # data = kohtaan datasetti, # FUN = kohtaan käytettävä funktio, jota käytetään.

Quantile fuktiolla saadaan laskettua kvantiili. Quantile funktiossa ensiksi valitaan data, josta kvantiili otetaan. Kohtaan seq( , ,kolmanteen väliin kvantiili, joka halutaan)

Apply funktiolla voidaan käyttää haluttua funktiota datasetin valittuihin riveihin tai sarakkeisiin. Apply funktiossa ensiksi datasetti, [hakasulkeissa] voidaan indeksien avulla määritellä # mitkä rivit tai sarakkeet datasta otetaan käyttöön. Menee aina näin: [rivit, sarakkeet] Nyt valittiin [2:3] komennolla 2. ja 3. sarake datasta Ekan pilkun jälkeen laitetaan 1 tai 2. Jos laitetaan 1, niin silloin jaotellaan data rivien mukaan. Jos taas 2, niin se jaotellaan sarakkeiden mukaan.

2.1.1 Teoriaa kvantiileista

Kvantiilit / persentiilit

  • x-Kvantiili kuvaa sitä pistettä jonka alle jää x prosenttiosuus ja ylle y prosenttiosuus havainnoista (y= 100%-x)

    • esim. mediaanin (50-persentiili) alle jää 50 % havainnoista ja ylle 50%

    • tai 90-persentiilin alle 90% ja ylle 10%

    • Esim.

      • Manuaalisesti saadaan haettua siten, että asetetaan luvut suuruusjärjestykseen ja etsitään oikea kohta. Esim 7 luvun sarjassa (1,2,3,4,5,6,7) mediaani on 4. Yläpuolelle jää 3 lukua ja alapuolelle 3.

      • 90.-persentiili sarjassa (1,2,3,4,5,6,7) asettuu lukujen 6 ja 7 välille. Kun luku ei osu suoraan kohdalle täytyy se interpoloida.

    • kvantiili kuvaa siis alueiden rajalla olevaa pistettä

  • R-kielellä kvantiilin saa etsittyä quantile() funktiolla

quantile(c(1,2,3,4,5,6,7), probs = seq(0, 1, 0.5), na.rm = FALSE,
         names = TRUE)
  0%  50% 100% 
   1    4    7 
    quantile(x, prob = seq(0, 1, 0.5), na.rm = FALSE, 
  names = TRUE, type = 7)

  • x = data/vektori, josta kvantiili otetaan

  • probs = seq(0, 1, 0.25) –> ekat kaksi numeroa pidetään aina 0 ja ykkösenä. Määrittää sekvenssin rajat. Viimeinen luku määrittää miten sekvenssi jaetaan

    • toisin sanottonu 0,25 kohtaan laitetaan haluttu kvantiili

      • 0,25 niin tulee persentiilit 25% välein

      • 0,90 tulee 90-persentiili ja 0,5 mediaani

  • na.rm = FALSE, saa tarvittaessa puuttuvat arvot jätettyä huomiotta. Tällöin merkattava TRUE, jolloin poistaa puuttuvat arvot.

  • names = TRUE, näyttää prosentit

  • type =, valitaan tyyppi, toimii ilmankin.

2.2.1 2. osan datansyöttötehtävä

Tehtävänanto:

Tee ohjelma, jolla voit syöttää muuttujalle arvot. Eli ohjelma ensin kysyy käyttäjältä kuinka monta havaintoa talletetaan ja sitten ohjelma kysyy jokaisen havainnon arvon. Ja nämä talletetaan data.frame:en. lopuksi skripti tulostaa datan. palauta R-skripti

Lisähaasteena voit tehdä niin, että ohjelma kysyy kuinka monta muuttujaa talletetaan, ja muuttujien nimet, ja sitten vuorotellen kaikille muuttujille havaintojen arvot. 

Tarpeellisia funktiota voi olla: readline() , esim: readline(prompt = “Anna havaintojen määrä:”) data.frame() , uuden data-kehikon luominen paste() , muuttujien yhdistäminen, esim. paste(“Anna havainnon”, i, “arvo:”) for-loop, esim. 

for (i in 1:num_obs) {
  ….
} names(data) , data-kehikon muuttujien nimet print()

Ratkaisu:

#kysy havaintojen lukumäärä 
nObs <-as.numeric(readline(prompt = "Anna havaintojen lukumäärä: "))

#kysy muuttujien lukumäärä 

MuutLkm <- as.numeric(readline(prompt = "Anna muuttujien lukumäärä: "))

#muuttujien nimet 

MuutNimet <- character(MuutLkm)

for (i in 1:MuutLkm){
  
  MuutNimet[i] <- readline( prompt = paste('Anna muuttujan',i, 'nimi: '))
  }

#luodaan datafreimi

datafreimi <- data.frame(matrix(nrow = nObs, ncol = MuutLkm,))
colnames(datafreimi) <- MuutNimet

#lisätään havaintoarvot datafremiin muuttuja kerrallaan

for (i in 1:MuutLkm){ 
  for (j in 1:nObs){
    datafreimi[j,i] <- readline(prompt = paste("anna muuttujan", MuutNimet[i], "havainnon", j, "arvo: "))  
    
  }
  
}

print(datafreimi)

2.3 Funktiot: readline(), paste(), data.frame(), for loop(){}, as.numeric(), numeric(), character(), matrix()

nObs <- as.numeric(readline(prompt = "Anna havaintojen lukumäärä: "))

readline()

  • R-kielen komento, jonka avulla ohjelma pysähtyy ja odottaa, että käyttäjä kirjoittaa jotain näppäimistöllä konsoliin.

  • Kun R kohtaa readline()-komennon, se ei jatka koodin suorittamista ennen kuin painat Enter. Kaikki, mitä kirjoitat ennen Enteriä, tallennetaan muuttujaan.

  • Tärkeää muistaa: Se palauttaa aina tekstiä

    • Tämä on yleisin virhe readline()-funktion kanssa. Vaikka kirjoittaisit numeron (esim. 25), R tallentaa sen tekstinä (“25”), ei lukuna.

    • Jos haluat käyttää syötettä laskutoimituksissa, sinun täytyy muuttaa se numeroksi as.numeric()-funktiolla:

prompt =

  • On tärkeää huomata, että R-kielessä prompt ei ole itsenäinen “komento” samalla tavalla kuin vaikkapa print(), vaan se on argumentti (eli lisäasetus), jota käytetään muiden funktioiden sisällä.

  • Yleisimmin esiintyy readline()-funktion yhteydessä.

  • Prompt tarkoittaa suomeksi kehotetta. Sen tehtävä on kertoa käyttäjälle, mitä hänen odotetaan tekevän. Se tulostaa tekstin konsoliin ja jättää kursorin odottamaan käyttäjän kirjoitusta.

    Ilman prompt-asetusta tietokone vain pysähtyisi odottamaan syötettä, ja käyttäjä olisi ihmeissään, miksi mitään ei tapahdu.

as.numeric()

  • muuttaa syötteen tekstista numeeriseksi

  • tämän myötä esimerkiksi laskutoimitukset onnistuvat

numeric(5)
[1] 0 0 0 0 0

numeric()

  • Luo tyhjän numeerisen vektorin, jonka voi täyttää

  • Sulkuihin voi määritellä montako paikkaa vektoriin tulee

    • esim. numeric(5) luo vektorin, jossa 5 nollaa eli 5 tyhjää paikkaa täytettäväksi
    • voi laittaa myös muuttujan sulkeisiin, jolloin paikkoja tulee muuttujan arvon verran
character(6)
[1] "" "" "" "" "" ""

character()

  • Luo tyhjän tekstipohjaisen “vektorin” / listan, jossa ennalta määritelty määrä paikkoja

  • esim character(6) vektorissa 6 paikkaa

  • Sulkuihin voi laittaa myös muuttujan, jolloin paikkojen lukumäärä tulee muuttujan arvon mukaan esim. character(Muuttuja)

datafreimi <- data.frame(matrix(nrow = nObs, ncol = MuutLkm,))
colnames(datafreimi) <- MuutNimet
