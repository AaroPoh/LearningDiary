9  Datasetin luominen

9.1 rnorm()

rnorm(n, mean, sd)

  • luo normaalijakaumaa noudattavan vektorin, jossa

    • n —> lukumäärä

    • mean —> keskiarvo (oletuksena 0)

    • sd —> keskihajonta (oletuksena 1)

  • Esim. alla luodaan normaalijakaumaa noudattava 10 alkion vektori, jonka keskiarvo on 6 ja keskihajonta 2.

rnorm(10, 6, 2) 
 [1] 7.044504 6.862561 5.501062 5.831666 7.147221 2.882727 9.171874 3.784720
 [9] 5.834423 3.857053

Tarkistetaan, että keskiarvo ja keskihajonta pitävät paikkansa.

vekt10 <- rnorm(10,6,2) 
vekt100 <- rnorm(100,6,2) 
vekt1000 <- rnorm(1000,6,2) 

m1 <- mean(vekt10) 
sd1 <- sd(vekt10) 
m2 <- mean(vekt100) 
sd2 <- sd(vekt100) 
m3 <- mean(vekt1000) 
sd3 <- sd(vekt1000) 
RivNimet <- c("Keskiarvo", "Keskihajonta")

data.frame(N10 = c(m1,sd1), N100 = c(m2,sd2), N1000 = c(m3, sd3), row.names = RivNimet)
                  N10     N100    N1000
Keskiarvo    6.106365 5.918489 6.121000
Keskihajonta 1.182211 1.919874 2.025645

Isommalla n:llä näyttäisi osuvan vielä tarkemmin kohdalleen.

9.2 Data framen luominen

R1 <- rnorm(5,0)
R2 <- rnorm(5,1)

data <- data.frame(arvo = c(R1,R2), ryhmä = rep(c(1,2), each=5))
data
         arvo ryhmä
1   0.2525517     1
2   2.1341160     1
3  -1.4825342     1
4  -0.1207210     1
5   1.9155540     1
6   0.8296448     2
7  -0.1376527     2
8   1.8557234     2
9   1.4849202     2
10  0.4506254     2

data.frame(sarake1, sarake2, sarake3, …)

  • Eli sulkuihin erotellaan pilkuilla eri sarakkeisiin menevä data

  • sarakkeet pystyy nimeämään laittamalla arvojen eteen nimen ja = (esim. arvo=R1)

  • esimerkissä käytetty c() eli combine funktiota, joka yhdistää kahdesta vektorista yhden

    • c() funktiolla saa siis yhdistettyä useamman vektorin yhden sarakkeen alle

  • row.names= argumentilla saa rivit nimettyä

    • täytyy viitata vektoriin esim. ylemmässä keskiarvo ja keskihajonta tehtävässä luotiin kirjainvektori RivNimet, johon sitten viitattiin

      • RivNimet <- c(“Keskiarvo” , “Keskihajonta”)

9.3 Satunnaistaminen tai satunnaisnumero sample() funktiolla

sample(x, size, replace = FALSE, prob = NULL)

  • x -> datajoukko, josta numerot vedetään

    • esim. 1:10, generoi numeroita väliltä 1-10

    • voi myös viitata muuttujaan

  • size -> montako numeroa otetaan

    • esim 1 palauttaa yhden lukuarvon ja 2 2 lukuarvoa jne

  • replace = FALSE -> palautetaanko jo nostetut arvot koriin, jolloin niitä voidaan käyttää uudelleen vai käytetäänko jokaista arvoa maksimissaan kerran

    • oletuksena FALSE eli arvoja käytetään vain kerran

      • esim. permutaatiotestissä joka arvoa käyteetään vain kerran

    • jos vaihtaa TRUE niin arvoja voidaan käyttää useampaan otteeseen

      • bootstrap menetelmässä arvoja puolestaan käytetään useampaan otteeseseen

9.4 Uuden kansion luominen dir.create()

dir.create("uusi_kansio")

  • Luo uuden kansion työskentelykansioon

    • työskentelykansion voi aina tarkistaa getwd() funktiolla
dir.create("kurssi/viikko1", recursive = TRUE)

  • recursive = TRUE -> mahdollistaa yhdellä kerralla kansion ja alikansion luomisen

    • esimerkissä luo kansion kurssi ja siihen alikansion viikko1 kansion

Hyvä muistaa

  • Ei ylikirjoita olemassaolevien kansioiden päälle, vaan antaa varoituksen

  • varmista, että nojaviivat ovat eteenpäin, jos ei toimi

    • vain yksi kenoviiva kansioiden välille

9.5 Kansion luominen if-lauseen kanssa

#| eval: false 
uusi_paakansio <- "Koulutehtava"

if (!dir.exists(uusi_paakansio)) {
  dir.create(uusi_paakansio)
}

  • Kansio voi olla järkevää luoda if lausekkeen kanssa koska

    • se yrittää luoda kansion, vain jos sen nimistä kansiota ei vielä ole olemassa

    • Eli sen jälkeen, kun kansio on jo luotu, niin seuraavilla ajokerroilla ei yritetä turhaan luoda sitä uudestaan

      • olemassa olevalle kansiolle ei käy mitään uusilla ajokerroilla, vaikka sitä pyrittäisiin luomaan uudestaan, mutta se antaa virheilmoituksen/varoituksen konsoliin, joka tekee siitä sekavamman

  • ! <- merkki tarkoittaa loogista ei operaattoria

    • eli if lauseen ehto menee: jos uusi_paakansio nimistä kansiota ei ole olemassa niin tee seuraava asia

      • luo uusi kansio dir.create() funktiolla

    • ilman ! merkkiä ehto menisi siis: jos uusi_paakansio on olemassa niin tee seuraava asia

9.6 Tiedosto ja kansiopolun muuttaminen file.rename()

file.rename(from, to)

  • file.rename() funktiossa

    • from kohtaan alkuperäinen siirrettävän kansion nimi

    • to kohtaan uuden yläkansion nimi / siirrettävän kansion nimi

dir.create("kurssi/viikko1", recursive = TRUE)
dir.create("Salatut_elamat")

file.rename("kurssi","Salatut_elamat/kurssi")

Esim.

  • Luotiin kansio kurssi ja sen alakansio viikko1

  • Lisäksi luotiin kansio Salatut_elamat

  • file.rename() funktiolla saadaan siirrettyä kurssi kansio Salatut_elamat kansion alle

    • koko “kurssi” kansio siirtyy eli sen alla oleva viikko1 siirtyy samalla

9.7 Tiedostopolun kirjoittaminen file.path() funktiolla

file.path("tietokone", "käyttäjä", "omatkansiot", "data", "measures")

Kun kirjoittaa file.path() funktioon esimerkin mukaisesti R rakentaa sen näin:

  1. Ottaa sanan tietokone

  2. Lisää erottimen /

  3. Ottaa sanan käyttäjä

  4. Lisää erottimen / jne.

  • Eli file.path() funktiolla saa tiedostopolun helposti talteen ilman virheitä ja kenoviivojen kanssa sähläämistä
